스레드기반 프로젝트
현재 진행 중인 프로젝트에 스레드를 넣어서 진행을 할려고 했으나 하는 중에 많은 오류가 있어서 간단하게 차량 랜트, 반납을 보여주는 예시를 만들었습니다. 일단은 회원을 10명으로 설정을 한다음에 랜트를 완료하면 회원 회원 중 누가 랜트 중인지 출력이 되게 하였습니다.
차량은 간단하게 브랜드 3개로 지정을 하고 실행을 하였습니다.
